# ClipCraft

ClipCraft is a client-side tool to convert SVG paths into CSS `clip-path` polygons.
Built with React, Vite, and Tailwind CSS.

## Features

- **Client-side only**: No server uploads; all processing happens in your browser.
- **SVG Parsing**: Extracts paths from uploaded SVG files.
- **Visual Preview**: See the original SVG vs the clipped result.
- **Customizable**: Adjust tolerance and decimals for the generated polygon.
- **Code Generation**: Get ready-to-use CSS and SVG `<clipPath>` code.
- **Internationalization**: English and Ukrainian support.
- **Dark Mode**: Automatically adapts or toggles manually.

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
